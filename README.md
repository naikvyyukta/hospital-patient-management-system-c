# Hospital Patient Management System

This project is implemented using Queue (via Doubly Linked List) in C. It simulates how patients are managed in a hospital using FIFO (First In First Out) approach.

## Features
- Add patient (enqueue)
- Remove patient (dequeue)
- Display patient list

## Data Structure Used
Queue implemented using Doubly Linked List

## How to Run
1. Compile the program using a C compiler
2. Run the executable
3. Follow menu options
